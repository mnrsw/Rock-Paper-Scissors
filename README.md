# Rock-Paper-Scissors
Python Game named Rock-Paper-Scissors

# Installing

Install  `random` using `pip` in Terminal

Windows: `pip install random`
Mac: `pip3 install random`

# Controls

Press `1` for Rock 🪨
Press `2` for Paper 🧻
Press `3` for Scissors ✂️
